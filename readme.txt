== Description ==
This plugin relies on the API provided by: https://www.coinmarketcap.com

CoinMarketCap API documentation can be viewed here: https://coinmarketcap.com/api/

Quickly display the most current and up to date cryptocurrency prices on your website. 


== Installation ==
1. Upload/install the plugin on your WordPress website
2. Place the [CryptoRates] shortcode wherever you\'d like to display current coin rates
3. That\'s it!
